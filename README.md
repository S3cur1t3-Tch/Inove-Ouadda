# Inove-Ouadda
​Description Générale du Projet : INOVE-OUADDAÏ (Ouaddaï Smart Horizon) ​INOVE-OUADDAÏ est une initiative technologique et stratégique d'une durée de 5 ans, dotée d'un budget optimisé de 4,8 Milliards de FCFA (7,3 M€). Porté par une ambition de souveraineté économique, ce projet vise à transformer la province du Ouaddaï (République du Tchad)
